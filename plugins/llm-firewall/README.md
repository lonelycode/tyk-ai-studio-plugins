# LLM Firewall

Security firewall for LLM requests providing content filtering, prompt injection detection, and policy enforcement.

## Features

- Content filtering for requests and responses
- Prompt injection detection
- Policy enforcement
- Real-time security monitoring

## Installation

Pull the plugin using:

```
/plugins/llm-firewall:1.0
```

## Configuration

See the plugin configuration options in Tyk AI Studio after installation.
