# Changelog

All notable changes to the Echo Agent plugin will be documented in this file.

## [0.1.0] - 2025-11-04

### Added
- Initial release of Echo Agent plugin
- Configurable prefix and suffix for LLM response wrapping
- Support for custom LLM selection
- Model override capability
- Optional metadata inclusion
- LLM proxy service integration

### Features
- Wraps LLM responses with custom markers
- Fallback echo mode when no LLM is available
- Configuration via JSON schema
- Full AI Studio SDK integration
